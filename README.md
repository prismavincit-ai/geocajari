# GeoCajari

Geoportal municipal de Cajari (MA) com mapas de uso do solo, população, relevo, hidrografia e localidades.

## Publicação no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todo o conteúdo desta pasta (`PUBLICAR_GEOPORTAL`) para a raiz do repositório.
3. Em `Settings > Pages`, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. Aguarde o GitHub gerar o endereço público.
